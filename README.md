# Yorùbá OCR Research

Repository for experiments, analysis, and paper writing for Yorùbá OCR.

## Layout

- `data/`: raw inputs, processed artifacts, and split files; VL-1.5 SFT export at `data/paddleocr_vl15_sft/` (generated by `scripts/14_export_paddleocr_vl_sft.py`, gitignored when large)
- `experiments/`: checkpoint output for baseline, fine-tuned, ablation, and optional VL LoRA (all **gitignored** except `.gitkeep`; regenerate via training scripts)
- `paper/`: paper sections, figures, and bibliography
- `scripts/`: preprocessing, training, and evaluation scripts
- `results/tables/`: traceable metrics used in manuscript text
- `scripts/shell/`: phased bash runners (`run_all.sh`, Drive backup) — see `scripts/shell/README.md`
- `.cursorrules` and `.cursor/rules/`: Cursor project and context rules
